# Designer Dashboard

A centralized dashboard designed for stakeholders and clients, enabling effortless customization of web interfaces to meet unique requirements.

## Features

- **Black & Orange Theme** - Professional dark theme with vibrant orange accents
- **3D Animations** - Smooth scroll animations and hover effects using Framer Motion
- **Responsive Design** - Fully responsive across all devices
- **Next.js 15** - Built with the latest Next.js and React 19
- **Tailwind CSS v4** - Modern styling with the latest Tailwind

## Tech Stack

- Next.js 15.5.4
- React 19.1.0
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Turbopack

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build

```bash
npm run build
npm start
```
